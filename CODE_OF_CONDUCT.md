# Code de conduite

Open Community Chat souhaite offrir un environnement respectueux et accueillant.

## Attendus

- Respect des autres membres
- Bienveillance
- Entraide
- Esprit constructif

## Interdits

- Harcèlement
- Discrimination
- Insultes
- Menaces
- Comportements toxiques

Les mainteneurs se réservent le droit de supprimer tout contenu contraire à ces règles.
